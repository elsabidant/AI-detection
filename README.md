AI Text Detection Model

This project explores the detection of AI-generated text using a combination of linguistic features, text embeddings, and machine learning models.

Files
- '1. Data preparation (human).ipynb' – Notebook to retrieve a sample of paragraphs from the ten works.
- 2. AI model.ipynb – Notebook containing preprocessing, feature extraction, model training, and evaluation.
- victorian_texts_human.csv – human text samples.
- IA_texts.csv – AI-generated text samples.
